# My PC Assistant

Voice-controlled local assistant for automating system tasks.

## Features
- Open apps/files via voice
- LLM-based intent parsing
- Offline TTS

## Setup
```bash
pip install -r requirements.txt
python -m app.main
```

## Tech Stack
- Python
- Ollama (LLM)
- SpeechRecognition
- pyttsx3
